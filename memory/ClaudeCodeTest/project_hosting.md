ClaudeCodeTest 레포는 GitHub에 호스팅되어 있다.

**Why:** 사용자가 여러 서버에서 작업할 가능성이 있으며, 코드와 CLAUDE.md가 GitHub을 통해 동기화된다.

**How to apply:** 작업 완료 시 반드시 커밋 + 푸시하여 다른 환경에서도 최신 상태를 유지할 수 있도록 한다. 메모리/설정은 서버마다 별도이므로 중요한 컨텍스트는 CLAUDE.md에 기록하는 것이 좋다.
